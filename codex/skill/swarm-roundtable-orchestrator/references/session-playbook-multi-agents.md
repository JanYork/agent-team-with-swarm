# Session Playbook: Multi-Agent Roundtable + Swarm

## Session Header
- Mission
- Success criteria
- Deadline/timebox
- Constraints

## Activation Checklist
1. Confirm `collaboration preset` (`nexus_full` / `nexus_sprint` / `nexus_micro`).
2. Confirm `primary_mode` and `enabled_modes`.
3. Confirm expert roster and hard-veto roles.
4. Confirm phase gates and escalation chain.

## Canonical Flow
1. Divergent swarm: generate at least 2 options.
2. Roundtable critique: score options with shared criteria.
3. Debate judgement (if enabled): jury verdict with evidence.
4. Convergent execution: assign owners and finalize deliverables.
5. Reality gate: evidence-based release decision.

## Handoff Contract
- Each handoff must include:
  - Task reference
  - Inputs
  - Expected output schema
  - Acceptance criteria
  - Evidence requirement

## Escalation Policy
- Dev/QA loop max retries: 3
- On retry exhaustion: escalate to `agents_orchestrator` then `studio_producer`/coordinator
- No phase transition without gate pass

## Suggested Role Pack For Roundtable Swarm Demo
- `agents_orchestrator`
- `content_creator`
- `image_prompt_engineer`
- `social_media_strategist`
- `legal_compliance_checker`
- `analytics_reporter`
- `experiment_tracker`
- `evidence_collector`
- `reality_checker`
