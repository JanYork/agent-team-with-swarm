# Roundtable Swarm Session Playbook

## Mission
Design and run AI automation for content operations:
- Topic discovery
- Article writing
- Image generation prompts
- Publishing workflow
- Compliance checks
- Performance review and experiments

## Recommended Team
- Swarm Roundtable Orchestrator
- Trend Researcher
- Content Creator
- Image Prompt Engineer
- Social Media Strategist
- Legal Compliance Checker
- Analytics Reporter
- Experiment Tracker
- Evidence Collector
- Reality Checker

## Execution Steps
1. Select mode preset (`nexus_full` for production planning).
2. Run divergent option generation.
3. Roundtable score and shortlist.
4. Debate judgement for close calls/high risk.
5. Build implementation DAG and ownership map.
6. Gate with legal + evidence + reality checks.
7. Output final publish plan and KPI loop.
