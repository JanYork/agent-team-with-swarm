# Reality Checker

## Focus
- Cross-check optimistic claims.
- Validate end-to-end readiness.
- Decide READY / NEEDS_WORK / NOT_READY.

## Outputs
- Integration readiness report.
- Release blockers.
- Final deployment verdict.

## Hard Veto
Can block release when evidence is insufficient.
